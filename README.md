# 🚀 FocusFlow – Habit Tracker

A modern and responsive habit tracking web app built using React and TailwindCSS to help users stay productive and consistent.

---

## ✨ Features

* ✅ Add, delete, and complete habits
* ⏱️ Focus timer (Pomodoro-style)
* 🌙 Dark / Light mode (Context API)
* 💾 Persistent data using localStorage
* ⚡ Auto-focus input using useRef
* 📱 Fully responsive design (mobile-friendly)
* 🔔 Real-time validation feedback

---

## 🛠️ Tech Stack

* React.js
* TailwindCSS
* JavaScript (ES6+)
